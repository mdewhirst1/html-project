Css Animations

What are CSS Animations?

An animation lets an element change from one style to another.

You can change as many CSS properties you want, as many times you want.

To use CSS3 animation, you must first specify some keyframes for the animation.

The Basics

Keyframes hold what styles the element will have at certain times.

To specify the key frames use '@keyframes example'.

The keyframes then need to be linked to the element that you want to animate.

This is done by adding 'animation-name: example'.

You also need to add a duration for the animation using 'animation-duration:'.

Note the duration is set to 0 as default the resulting animation would have no effect.

Example

The following code changes the div element from red to yellow

/* The animation code */
@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}

/* The element to apply the animation to */
div {
  width: 100px;
  height: 100px;
  background-color: red;
  animation-name: example;
  animation-duration: 4s;
}

It is possible to set multiple key frames using % insted of from and to.

@keyframes example {
  0% {background-color: red;}
  25% {background-color: yellow;}
  50% {background-color: green;}
  75% {background-color: blue;}
  100% {background-color: red;}
}

Repeating an animation

An animation can be repeated using 'animation-iteration-count:'.

This can either be a number or infinite

Example

The following code uses "infinite" to make the animation repeat for ever.

/div {
  animation-name: example;
  animation-duration: 4s;
  animation-iteration-count: infinite;
}

Reversing and Alternating an animation

An animation can be to animate in revese using 'animation-direction: reverse;'.

This will cause the animation to run in reverse to what was specified.

Example

div {
  animation-name: example;
  animation-duration: 4s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

Or it can be set to alternate with 'animation-direction: alternate;'

This will cause the animation to run forward, then in reverse, then forward.

Example

div {
  animation-name: example;
  animation-duration: 4s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

The Speed Curve of the Animation

The 'animation-timing-function:' specifies the speed curve of the animation.

The animation-timing-function property can have the following values:

  • ease - specifies an animation with a slow start, then fast, then end slowly (this is default)
  • linear - specifies an animation with the same speed from start to end
  • ease-in - specifies an animation with a slow start
  • ease-out - specifies an animation with a slow end
  • ease-in-out - specifies an animation with a slow start and end
  • cubic-bezier(n,n,n,n) - lets you define your own values in a cubic-bezier function

Delaying an Animation

An animation can be delayed with 'animation-delay:'

Shorthand

The two examples below both produce the same result.

div {
  animation-name: example;
  animation-duration: 4s;
  animation-timing-function: linear;
  animation-delay: 2s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

div {
  animation: example 4s linear 2s infinite alternate;
}
